# SecPass
Secure web-based password manager

## Building from source
Requirements:
 - Rust >= Stable
 - Cargo >= Stable

Clone from Git
> git clone https://github.com/ltheinrich/secpass && cd secpass

Build using Cargo
> cargo build --release

Run the binary `./target/release/secpass`
